# Simple Binary Classification With QML
**(This project was done in february)**

This is a simple project that I made to get some experience with QML and learn some of the basics of quantum machine learning with Pennylane. This model uses a kaggle dataset for predicting diabetes using 8 features:
- \# of pregnancies
- Plasma glucose concentration
- Diastolic blood pressure
- Triceps skin fold thickness
- 2-Hour serum insulin
- Body mass index
- Diabetes pedigree function
- Age

The kaggle dataset can be found here: https://www.kaggle.com/datasets/girishvutukuri/diabetes-binary-classification
